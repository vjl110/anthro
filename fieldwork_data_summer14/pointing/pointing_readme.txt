point_twe file contains IDs, Point of Origin (PoO) and degrees pointed to target

point_coordinates file contains GPS coordinates for all PoO and targets included in the study.

BYROW is the same data from point_twe, but it is organized such that each row is an individual point event.  This file also has information for the point_coordinates added in.

Participant points recorded using a Brunton compass. Declination set to -9.5 degrees
